<h2>Installazione</h2>
Apri un terminale ed esegui questi comandi

```sh
git clone https://github.com/UsboKirishima/lived.git

cd lived

node src/index.js
```

<h2>Aggiornamento</h2>
Esegui lo script update.sh

```sh
chmod +x update.sh

bash update.sh
```
Or
```sh
git pull
```
