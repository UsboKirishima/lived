<h2>Installation</h2>
Open the terminal and run following commands

```sh
git clone https://github.com/UsboKirishima/lived.git

cd lived

node src/index.js
```

<h2>Update</h2>
Run the update.sh script

```sh
chmod +x update.sh

bash update.sh
```
Or
```sh
git pull
```
